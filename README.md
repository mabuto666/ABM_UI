# ABM_UI

Standalone, reusable telemetry consumer UI using a pluggable adapter model.
This repo includes an ABM filesystem adapter, but the UI is designed for reuse by other projects.

## Run locally
```bash
cd web
python3 -m http.server 8000
# open http://localhost:8000

## Demo
```bash
./demo.sh
# open http://localhost:8000
```
